# AI Education App

AI教育向けの静的Webアプリです。初級コース30分×10回、上級コース30分×10回の学習計画をブラウザで確認できます。

## できること

- 初級コースと上級コースの切り替え表示
- 各回のテーマと到達目標の確認
- 完了チェックによる進捗管理
- ブラウザ内への進捗保存

## ローカルで開く方法

index.html をブラウザで開くだけで利用できます。

## GitHub Pages で公開する方法

1. この内容を GitHub の公開リポジトリに push します。
2. GitHub のリポジトリ画面で Settings を開きます。
3. Pages を開き、Source を Deploy from a branch にします。
4. Branch は main、フォルダは /(root) を選んで保存します。
5. 数分後に公開URLが発行されます。

## ファイル構成

- index.html: 画面構成
- styles.css: デザイン
- script.js: カリキュラム表示と進捗管理
